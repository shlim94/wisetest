Project Description

The $rootnamespace$ project is an XAF Middle Tier application server implemented
as a Windows Service. Modify the app.config file to setup data connection 
for server and client. Edit the ServerApplication.cs(vb) file to add client 
application’s modules. Set up the server-side security in the 
ApplicationServerService.cs(vb) file. Design the ProjectInstaller.cs(vb) file 
to configure service installation.
